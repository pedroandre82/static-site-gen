# Static Site Generator in Python

## 📖 Project Overview
This project is a **static site generator** built from scratch in Python. A static site generator takes raw content files (like Markdown and images) and transforms them into a fully functional static website composed of HTML and CSS. 

Static sites are widely used for blogs, portfolios, landing pages, and documentation because they are:
- ⚡ Lightning-fast
- 🔒 Secure
- 🌍 Easy to host

## 🎯 Purpose
The goal of this project is to learn the fundamentals of static site generation by building the tool that creates the site itself. It’s a meta-project: instead of just building a website, you’ll build the generator that produces websites.

## 🚀 Features
- Converts Markdown content into HTML
- Supports images and other static assets
- Produces clean, static files ready for deployment
- Ideal for content-heavy sites like blogs or documentation

## 🛠️ Tech Stack
- **Language:** Python
- **Content Format:** Markdown
- **Output:** HTML + CSS

## 📌 Use Cases
- Personal blogs
- Developer portfolios
- Landing pages
- Project documentation
